# ParaPaint
Pixel art software

## Описание
Программа для создания пиксель арта.
Выбор цвета с помощью oklab.

## Управление
Левая кнопка мыши — использует кисть или другой выбранный инструмент. При наведении на выделенные пиксели — перетаскивать при удержании кнопки.

Правая кнопка мыши — Пипетка. Если существует выделение, то отмена выделения.

Зажатая правая кнопка мыши — делает прямоугольное выделение.

Средняя кнопка (колёсико) — При удержании перемещение по холсту.

Промотка колёсиком — перемещение по холсту по вертикали.

Промотка колёсиком с зажатым Shift — перемещение по холсту по горизонтали.

A — Уменьшить кисть.

D — Увеличить кисть.

W — Приблизить холст.

S — Отдалить холст.

Z — Отмена последнего действия.

Y — Отмена отмены.

X — Переключение между первым и вторым выбранными цветами.

K — Обрезать холст до размеров текущего выделения, если оно есть.

Shift + K — Открывается окно, где можно изменить размер холста.

R — Открывается окно, где можно изменить размер изображения.

Tab — Скрыть интерфейс. Если скрыт, то показать интерфейс.

CTRL + C — Скопировать выделенное. Если ничего не выделено, то скопировать всё на слое.

CTRL + V — Вставить.

CTRL + A — Выделить весь холст.

CTRL + D — Отменить выделение.

Delete — Удалить выделенное.

CTRL + E — Свести выбранный слой со слоем ниже.

Цифра 1 — Кисть.

Цифра 2 — Заливка.
Цифра 3 — Замена цвета.
Цифра 4 — Прямая линия
Цифра 5 — Выделение пикселей такого же цвета рядом с указанным пикселем.
